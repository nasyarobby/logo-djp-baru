# Logo DJP Baru

Repository ini menyimpan Logo DJP baru dalam berbagai format untuk kepentingan pengembangan aplikasi atau website.

Pembuatan file - file mengutamakan optimasi ukuran file, dengan tetap memperhatikan

## Penjelasan direktori

Tiap direktori menyimpan format tertentu untuk tujuan tertentu

- Root

  `Logo-semua-komposisi.svg` adalah file yang mencakup semua logo dengan semua komposisi.

- SVG

  File dalam format SVG. Untuk dibuka dan diedit dengan menggunakan aplikasi Inkscape.

- PLAIN-SVG

  File dalam format SVG dengan elemen-elemen identik dengan aplikasi Inkscape sudah dihapus. Untuk diembed pada dokumen, contohnya HTML.

- PNG

  File dalam format PNG, non interlaced. Dikonversi dari file dari direktori SVG dengan DPI 96.

- WEBP

  File dalam format WEBP. Dikonversi dari file darri direktori PNG dengan menggunakan libwebp dengan parameter kualitas 80.
